## 🛠️ Source Code & Documentation

### Overview
An interactive budget planning tool that lets you track income and expenses with live-updating financial summaries and wealth projections.

### How to Run
1. Click the live demo link above, or
2. Open `Budget Calculator.html` directly in any modern web browser

### Features
- **Income Slider** — Adjust monthly income ($0–$20,000)
- **5 Expense Categories** — Housing, Groceries, Utilities, Entertainment, Savings (each $0–$3,000)
- **Live Totals** — Pie chart, total expenses, net savings, and savings rate update instantly
- **Budget Utilization Bar** — Visual indicator of how much of your income is spent
- **Future Wealth Predictor** — Projects savings over 1 year, 5 years, or a custom period (1–30 years)
- **Financial Health Tips** — Dynamic advice based on your savings rate
- **Interactive Sliders** — Drag sliders or type exact values in number inputs; both sync in real time
