## 🛠️ Source Code & Documentation

### Overview
A classic Snake game built entirely with vanilla HTML, CSS, and JavaScript — no frameworks or libraries required.

### How to Run
1. Click the live demo link above, or
2. Open `snake-game.html` directly in any modern web browser

### Features
- **Score Tracking** — Current score and all-time high score (saved in localStorage)
- **Difficulty Levels** — Easy (200ms), Medium (120ms), Hard (70ms) intervals
- **Keyboard Controls** — Arrow keys or WASD
- **Touch Controls** — On-screen D-pad for mobile/tablet
- **Game Over Modal** — Displays final score with "New High Score" badge
- **Responsive Design** — Adapts to mobile screens

### Controls
| Action | Keyboard | On-Screen |
|--------|----------|-----------|
| Move Up | ↑ or W | D-pad ↑ |
| Move Down | ↓ or S | D-pad ↓ |
| Move Left | ← or A | D-pad ← |
| Move Right | → or D | D-pad → |
| Pause/Resume | Space or Esc | Start/Pause button |
