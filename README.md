## 🚀 Quick Start

Each assignment is self-contained in its own folder. Open any HTML file directly in your browser, or use the live demo links above.

> ⚠️ **Live demos require GitHub Pages.** Go to **Repo Settings → Pages → set Branch to `main` and Folder to `/ (root)` → Save**. After ~2 minutes, all links above will be active.
